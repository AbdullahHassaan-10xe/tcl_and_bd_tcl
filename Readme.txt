For Running the project via tcl script:
1. Open Vivado TCL Console.
2. cd to the directory "work" inside fpga folder.
3. Then, run "source ../bram_project.tcl" to create the project.
4. cd to the directory "bd" inside fpga folder.
5. Run "source ../bram_bd.tcl", to create block design
6. Then Launch SDK.
